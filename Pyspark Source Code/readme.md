# Takeo Capstone Project - PySpark Source Code Repo

This repository contains the implementation artifacts for the Health Care Insurance Big Data project.

## Tech Stack
- AWS S3
- AWS Glue
- PySpark
- Snowflake
- GitHub

## Repository Structure
- `data_cleansing/` - AWS Glue ETL and data cleansing logic
- `sql_queries/` - Snowflake SQL for analytics

## Implementation Note
The original requirement mentioned AWS Redshift and Databricks.
In this implementation:
- Snowflake was used as the warehouse layer
- AWS Glue was used for ETL and cleansing
