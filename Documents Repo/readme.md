# Takeo Capstone Project - Documents Repo

This repository contains the documentation and presentation deliverables for the Health Care Insurance Big Data project.

## Contents
- Requirement Specification
- Solution Design
- Sample dataset
- Presentation 
- Screenshots

## Implementation Note
The original requirement mentioned AWS Redshift, Databricks, and Jira.
In this implementation:
- Snowflake was used instead of Redshift
- AWS Glue was used instead of Databricks
